Simple select

This is a simple HTML select element wrapped in dd-select. Most basic usage. You can pass a model, set a field on the model as target field, and pass a list of selectable items.

In this example we have an address model with a country_id field and a set of countries. The user can select only one country.

Usage:

    <dd-select
      mode="simple"
      [model]="address"
      field="country_id"
      [items]="countries"
    ></dd-select>

Example:

Model:

{{ address | json | prettyJson }}
