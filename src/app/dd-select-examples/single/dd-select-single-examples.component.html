Single select with dialog

This is a single select example with dialog. Dialog shows up if you click on select field. In the dialog we present a list component where the user can select only one items.

In this example we have an address model with a country field and a set of countries. The user can select one country.

The dialogSettings contains a dialog configuration to show CountryListComponent when shows the dialog.

Usage:

    <dd-select
      mode="single"
      [model]="address"
      field="country_id"
      [dialogSettings]="dialogSettings"
    ></dd-select>
